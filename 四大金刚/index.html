BOSS 3

实现一个简单的图片轮播。（可以使用jQuery, 但是不能用现有的插件， 当然你可以自己写一个）

提示： 如何使用setTimeout? 可以去github找一些源码看看 在index放在全局作用域中 动画可以使用jquery 的animate方法， 方便快捷 停止setTimeout 可以使用clearTimeout

可以采用多用方式初始化图片轮播

  1. jquery 插件的方式(比如unslider.js)
    var slider = $("#slider").unslider({
        speed: 2000,               //  The speed to animate each slide (in milliseconds)
        delay: 3000,              //  The delay between slide animations (in milliseconds
        keys: true,               //  Enable keyboard (left, right) arrow shortcuts
        fluid: false              //  Support responsive design. May break non-responsive designs
    });
  1. 闭包的方式(swipe.js)
    var mySwipe \= Swipe\(elem\, \{
        startSlide: 4,
        auto: 3000,
        continuous: true,
        disableScroll: true,
        stopPropagation: true,
        callback: function(index, element) {},
        transitionEnd: function(index, element) {}
    });
  1. 构造函数的方式
var runner = new Splash(document.getElementById('container'),{
    speed : 400,
    duration : 2000
});

要求： 切换的初始化函数返回一个图片切换对象的引用， 改对象拥有next（向右）, prev（向左）, stop（停止）, start（开始） 等方法

如:

    slider.next() // 轮播向右
    slider.prev() // 轮播向左
    slider.stop() // 轮播停止
    slider.start() // 轮播开始
